A web game inspired by Carmen Sandiego using JavaScript, MapBox and Firebase. The user must use clues given to them in the form of pictures taken around the world and various facts given to them to try and track down the criminal on the world map before time runs out.

Pill-Guru -

A medication organizer and tracker. Using MySQL, Node, API calls, and Express my team built an app that can track medication for a given user, store images of the given medication and alert the user to any possible conflicts among a medication list


* Authentication has not been completed *



You Brew -

An internal facing tracking app for Arches Brewing, Atlanta GA. This application was built using React, Node, Axios and MongoDB. This app allows the sales team in the field to have a live feed into the inventory of the brewery and their development cycle. The user can track processes, request new batches, reserve portions of incomplete batches for future sales and communicate with the brewers in real time.


* This has not been published publicly, but the source code is available *
